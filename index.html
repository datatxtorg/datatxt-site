Web Data Discovery

The robots.txt and humans.txt files decribe a website content for web crawlers and people. In a similar way, a data.txt file descibes a website content for data discovery and access. A data.txt file is just a yaml file with a list of attributes that describe one or many data sources. The only required attribute is a 'path' to point to data sources available in the website, but can also include many other optional fields like 'description', 'license', 'name', and so on.
